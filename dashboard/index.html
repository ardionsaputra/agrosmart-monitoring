Cara Kerja Sistem

Sensor membaca kondisi tanah setiap 5 detik. Jika kelembapan di bawah batas minimum, mikrokontroler mengaktifkan pompa air secara otomatis melalui relay.

Komponen Alat
Sensor Kelembapan: Capacitive soil moisture sensor
Sensor Suhu: DHT22
Mikrokontroler: ESP32 + WiFi
Aktuator: Pompa air mini + relay module
Display: Web dashboard real-time via MQTT
Tips Perawatan Sensor
  • Tanamkan sensor minimal 5 cm ke dalam tanah agar pembacaan akurat.
  • Hindari sinar matahari langsung pada sensor suhu DHT22.
  • Bersihkan ujung sensor setiap 2 minggu agar tidak ada kerak tanah.
  • Pastikan koneksi kabel aman dari air saat pompa aktif.
  • Kalibrasi sensor setiap bulan dengan membandingkan hasil manual.
